# 🎓 AI Code Mentor

An AI-powered tool that helps beginners understand code through intelligent explanations.

## Features
- 📝 Line-by-line code explanations
- 💬 Ask follow-up questions
- 🌍 Support for 14+ programming languages
- 🎯 4 different explanation styles

## Built for AMD Hackathon 2026

## How to Use
1. Enter your Groq API key (free from console.groq.com)
2. Paste any code
3. Get AI-powered explanations!
